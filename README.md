# Celebal Tech Assignment 1 - Pandas Data Cleaning Project

## About

This project is submitted by Harsh Vishwakarma from MMDU as part of the Celebal Tech assignment.

The project demonstrates basic data analysis and data cleaning operations using the Pandas library in Python.

---

## Technologies Used

- Python
- Pandas

---

## Tasks Performed

### 1. Loading Dataset

- Loaded CSV dataset using Pandas DataFrame.

### 2. Exploring Dataset

Used different Pandas functions such as:

- `head()`
- `tail()`
- `shape`
- `columns`
- `info()`

to understand the structure of the dataset.

### 3. Handling Missing Values

Handled missing values using:

- `fillna()`
- Dictionary mapping for multiple columns

### 4. Filtering and Selecting Data

- Selected required columns
- Filtered products with rating greater than or equal to 4.5

### 5. Detecting Duplicates

- Used `duplicated()` function to identify duplicate rows

### 6. Feature Engineering

Created a new column:

- `Top_rated_product`

based on product ratings.

### 7. Exporting Cleaned Dataset

Exported the cleaned dataset into a new CSV file.
