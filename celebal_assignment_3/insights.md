## Key SQL Learnings

During this project, the following SQL concepts were used:

- Database normalization using separate customer, product, and order tables
- Data cleaning and transformation using SQL
- Subqueries for advanced filtering
- Common Table Expressions (CTEs) for organizing complex queries
- Window Functions for ranking and customer analysis
- Joins for combining data from multiple tables

---

## Conclusion

This project demonstrates how SQL can be used to convert raw sales data into meaningful business insights.

By using data normalization, aggregations, subqueries, CTEs, joins, and window functions, it was possible to analyze customer behavior, identify top-performing customers, and understand overall sales patterns within the Superstore dataset.
